Conceptos básicos

Antes de hablar de dispositivos, configuraciones o antenas, conviene entender qué hay debajo de Meshtastic y por qué funciona como funciona.

Este apartado es corto pero clave. Si esto se entiende, todo lo demás encaja.

LoRa: la base de todo​

Meshtastic funciona sobre LoRa (Long Range), una tecnología de radio diseñada para:

  • transmitir pocos datos
  • a muy larga distancia
  • con consumo energético mínimo

No está pensada para velocidad ni para voz.
Está pensada para llegar lejos y durar mucho.

En entornos abiertos, una transmisión LoRa puede recorrer:

  • varios kilómetros entre nodos
  • decenas de kilómetros con buena antena y altura

Qué significa “mesh”​

Una red mesh (malla) no depende de un punto central.

Cada nodo:

  • envía sus propios mensajes
  • retransmite los mensajes de otros nodos

Esto permite que:

  • el alcance crezca con cada nodo añadido
  • la red siga funcionando aunque uno caiga
  • no haya un “servidor” que se pueda romper

En ruta, esto es clave:
si dos motos no se ven entre sí, otra puede hacer de puente.

Nodos: qué son exactamente​

Un nodo Meshtastic es un pequeño dispositivo que integra:

  • un microcontrolador
  • un chip LoRa
  • normalmente GPS
  • una antena
  • batería o alimentación externa

El nodo es el que:

  • habla por radio con otros nodos
  • se conecta por Bluetooth al móvil
  • gestiona mensajes y posición

El móvil no transmite por radio:
solo es la interfaz de control.

El móvil: control, no comunicación​

El móvil:

  • se conecta al nodo por Bluetooth
  • muestra mensajes
  • muestra posiciones
  • permite configurar el dispositivo

Si el móvil se queda sin cobertura:

  • no pasa nada
  • Meshtastic sigue funcionando

Si el móvil se apaga:

  • el nodo sigue activo
  • la red sigue operando

Canales: redes dentro de la red​

Meshtastic usa canales para separar grupos.

Un canal define:

  • quién puede leer los mensajes
  • quién puede enviar
  • si va cifrado o no

Ejemplo práctico:

  • un canal para el grupo principal
  • otro para pruebas
  • otro para emergencias

Sin el canal correcto, no ves ni envías nada, aunque estés cerca.

Mensajes y posiciones​

Meshtastic permite:

  • mensajes cortos de texto
  • envío de posición GPS
  • balizas periódicas

No es WhatsApp:

  • no hay fotos
  • no hay audios
  • no hay historial largo

Es comunicación funcional, no social.

Alcance real (sin magia)​

El alcance depende de:

  • antena
  • altura
  • entorno
  • potencia configurada
  • número de nodos

En moto:

  • valle cerrado → alcance corto
  • crestas y pistas altas → alcance enorme
  • ciudad → muy limitado

Por eso la planificación y el uso inteligente importan más que el hardware.

Lo importante a entender​

Antes de seguir, quédate con esto:

  • Meshtastic no es para hablar
  • Meshtastic no sustituye un intercom
  • Meshtastic no necesita cobertura
  • Meshtastic escala con el grupo
  • Meshtastic premia la preparación

Regla básica​

Cuantos más nodos bien colocados, mejor funciona la red.
Meshtastic no es magia, es radio.
